# Eclipse ([retour](../SOFTWARE.md))

Il s'agit d'un IDE complet permettant la programmation en Java. Personnellement, j'utilise la version Entreprise Java/Web Developers.

### Installation des prérequis :
```
sudo pacman -S webkit2gtk-4.1
```

### Téléchargement de l'installateur : 
https://www.eclipse.org/downloads/packages/

### Extraction et installation :
```
tar -xvzf eclipse-inst-jre-linux64.tar.gz
cd eclipse-installer
./eclipse-inst
```
