# Python et IDLE ([retour](../SOFTWARE.md))

IDLE est un IDE avec interface graphique permettant la programmation en Python.

Installation :
```
pacman -S python python-pip tk
```

Entrée Desktop IDLE :
```
[Desktop Entry]
Type=Application
Version=8.6.16
Name=IDLE
Comment=IDE Python 3.14.4
Path=/usr/bin
Exec=idle
Icon=/home/<username>/Images/Icons/python-idle-icon.png
Terminal=false
Categories=Development
```
