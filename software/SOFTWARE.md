# Logiciels ([retour](../README.md))

Il s'agit de répertorier les procédures d'installation des logiciels que j'ai personnellement installés. Cette partie est plutôt personnelle, car évidemment, tout va dépendre de vos besoins.

Toutefois, si des logiciels vous intéressent, sachez que les procédures ont fonctionné pour mon cas. Mais il se peut que d'ici-là, ces procédures soit dépassés.

Je tenterais de les mettre à jour, mais si besoin, vous pourrez rechercher le logiciel voulu soit dans la [documentation d'Arch](https://wiki.archlinux.org/title/Main_page), soit sur le site/dans la documentation officielle du logiciel.
