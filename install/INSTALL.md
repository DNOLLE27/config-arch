# Installation d'Arch ([retour](../README.md))

Contrairement aux autres distributions Linux, l'installation d'Arch se fait manuellement (configuration du support d'installation, partitionnement, installation du système de base, choix et installation du noyau, ...).

La procédure d'installation est décrite dans la [documentation officielle](https://wiki.archlinux.org/title/Installation_guide). Toutefois, dans le cadre de ce ReadMe, uniquement l'essentiel a été pris en compte.
